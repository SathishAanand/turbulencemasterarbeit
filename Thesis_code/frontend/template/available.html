Available models that can be used for this tool

The hyperparameters that can be chosen are Seqlength, Epochs, Batch Size, Learning Rate and Activation Function

The values used for trained models were


  • Seqlength -- 3ts, 5ts, 10ts, 20ts
  • Epochs -- 10, 20, 40, 80
  • Batch Size -- 128, 256
  • Learning Rate -- 0.01, 0.001, 0.0001
  • Activation Function -- Relu (1), Sigmoid (2), tanh (3)

Each timestep has 72 unique trained models

The file names are named for easy identification of the configurations. Seqlength --> sXXts, Epochs --> ep, Batch Size --> bs, Learning Rate --> lr and Activation Function --> af
There are total 288 different files available to experiment
For e.g. "turbulence37_s3tsep20bs256lr0.001af3" indicates that it is 7th file with 3 timesteps, 20 epochs, 256 batch size, 0.001 learning rate and tanh activation function
For e.g. "turbulence104_s10tsep10bs128lr0.0001af1" indicates that it is 4th file with 10 timesteps, 10 epochs, 128 batch size, 0.0001 learning rate and Relu activation function

tSNE Projections
The colorful dots in the tsne projections represents the timesteps. Based on the no.of.timesteps which have been selected, you can find the no.of.dots. For instance, if you have selected 3 timestep input, you will find 3 dots
The lines in the tsne projections represents the line(edge) between two timesteps. d3.interpolateRdYlGn is used as color coding for the lines.
Dimensionality Reductions
There are 3 dimesnionality reduction techniques in which the data cn be visualised
  1. tSNE
  2. PCA
  3. UMAP
K-Means Clustering
K-means clustering algortihm was implemented and by selecting the respective cluster checkboxes and the cluster button, the projection for the respective data can be visualised
Animation & Scatterpoints
By checking the animate and scatterpoints checkboxes and clcicking the buttons, the data can be visualised in animation and scatterpoints. d3.interpolateRainbow is used as color coding for scatterpoints.
